# bce_interest_rates_code

This repository contains Python code that scraps interest rates data from Ecuador
Central Bank (BCE). 

## Overview
The enters the BCE webpage monthly interest rates figures and retrieves the data.

Example link for January 2026: 
'https://contenido.bce.fin.ec/documentos/Estadisticas/SectorMonFin/TasasInteres/Indice.htm'


## Requirements
- Python

## Warning
Please note that the table structure is not constant throughout all periods.
